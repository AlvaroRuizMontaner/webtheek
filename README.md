## Funcionamiento de Stripe

En la pantalla de perfil/prices lo que hace el front es usar la libreria de Stripe y una public key (cuidado de no usar la secret key, que tambien sirve pero es peligroso) para consultar los productos y obtener los precios y los nombres. Una vez hecho eso, si se pincha en el botón de ButtonCheckout, lo que hace es llamar al servidor proporcionando los datos del producto que se quiere comprar (precio, nombre, etc) entonces el servidor usa esa misma librería y la secret key para cargar una sesión en stripe, y si sale bien, stripe le devuelve al back una url que luego es enviada al front, que al llegar alli redirige y lleva a la pantalla.
